# Query Feature Parser

A Chrome browser extension for parsing and managing URL query parameters.

## Features

- Parse query parameters from URLs
- Display parameters in a user-friendly interface
- Easy copy/paste functionality
- Real-time updates

## Development

This is a Chrome extension built with Manifest V3.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this directory
